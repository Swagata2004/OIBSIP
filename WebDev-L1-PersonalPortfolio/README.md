# Swagata Pal Portfolio

A modern, responsive personal portfolio website built as a digital portfolio and online resume for Swagata Pal. The site highlights skills, projects, contact details, and a downloadable resume in a polished one-page layout.

## Overview

This project is designed to present professional identity in a clear and visually engaging way. It combines a strong hero section, skill showcase, project cards, and contact information into a single static website that works well on desktop and mobile.

## Features

- Professional hero section with profile image and personal branding
- Sticky navigation with smooth scrolling
- GitHub shortcut in the navbar
- Resume button that opens and downloads the resume PDF
- About Me section with a short professional introduction
- Skills grid with icons for technical and creative tools
- Project showcase with three featured UI concepts
- Horizontal project navigation with scroll controls
- Contact section with name, email, phone number, and social icons
- Fully responsive layout for desktop and mobile
- Animated visual accents for a more premium presentation

## How to Run Locally

1. Download or clone the project files.
2. Open `index.html` in any modern browser.
3. For the best experience, keep the `assets` folder in the same directory as the HTML file.

## How to Use

- Use the navigation bar to jump between sections.
- Click **GitHub** to visit the GitHub profile.
- Click **Resume** to open and download the resume PDF.
- Use the **View Project** buttons and project arrows to browse the featured project cards.

## Deployment

This is a static website, so it can be deployed easily on:

- GitHub Pages
- Netlify
- Vercel
- Any static hosting service

### GitHub Pages

1. Push the repository to GitHub.
2. Go to repository settings.
3. Open the Pages section.
4. Select the main branch and root folder.
5. Save and wait for the site to publish.

## Customization

You can update the following quickly:

- Name and branding text in `index.html`
- Profile photo inside `assets/profile.jpg`
- Project preview images in the `assets` folder
- Resume PDF in `assets/Swagata-Pal-Resume.pdf`
- Colors, spacing, and layout in `styles.css`
