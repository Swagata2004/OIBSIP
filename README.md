🚀 Projects Overview:--

1. Apexora Landing Page
Folder: WebDev-L1-LandingPage

A fully responsive landing page for Apexora — a fictional tech brand. Designed to capture user attention with a modern layout and smooth navigation.

Sections Included:

🏠 Home – Hero section with call-to-action
ℹ️ About – Company overview and mission
🛠️ Services – Core offerings and capabilities
🖼️ Portfolio – Showcase of past work
📝 Blog – Latest articles and insights
💰 Pricing – Transparent pricing tiers
👥 Careers – Job openings and culture
📞 Contact – Get in touch form and details
Tech Stack: HTML5, CSS3, JavaScript (Vanilla)

2. My Digital Portfolio
Folder: WebDev-L1-PersonalPortfolio

A personal digital portfolio designed to highlight professional skills, projects, and achievements. Ideal for developers, designers, and creatives.

Sections Included:

🏠 Home – Introduction and tagline
👤 About – Background and bio
🧠 Skills – Technical and soft skills
🗂️ Projects – Featured work with descriptions
📬 Contact – Inquiry form
🐙 GitHub Button – Direct link to GitHub profile
📄 Resume Button – Download or view resume
Tech Stack: HTML5, CSS3, JavaScript (Vanilla)

3. Temperature Converter
Folder: WebDev-L1-TemperatureConverter

An interactive web tool for converting temperatures between three major units in real time.

Features:--

🌡️ Input fields for Celsius, Kelvin, and Fahrenheit
🔁 Dynamic, real-time conversion as you type
🎯 Three simultaneous output displays
📱 Responsive and mobile-friendly design
Tech Stack: HTML5, CSS3, JavaScript (Vanilla)
